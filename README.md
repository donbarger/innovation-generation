## Quick Start

### 1. Start the Backend

From the project root:

```bash
source .venv-backend/bin/activate
uvicorn backend.app:app --reload --host 0.0.0.0 --port 8000
```

The API will be available at `http://localhost:8000`.

### 2. Serve the Frontend

In another terminal:

```bash
cd frontend
python3 -m http.server 5500
```

Then open your browser to **http://localhost:5500**

Or simply double-click `frontend/index.html` (with CORS limitations).

## UI Sections

### Generation Panel
- Paste any YouTube URL or the URL to an online article
- Real-time progress updates with emojis
- Status colors: blue (running), green (success), red (error)

### Transcripts
- Shows transcripts extracted from videos or articles
- Useful for reference

### Innovations (Articles)
- Full-length innovation articles generated by PredictionGuard
- Structured with Key Insights, main content, reflection questions

### Substack Notes
- 2 punchy social media notes per innovation
- Ready to copy/paste to Twitter, LinkedIn, etc.

## API Endpoints

- `POST /api/generate` - Start a generation job
- `GET /api/jobs/{job_id}` - Check job status and progress
- `GET /api/transcripts` - List all transcripts
- `GET /api/innovations` - List all innovations
- `GET /api/substack_notes` - List all Substack notes

## Troubleshooting

**Backend won't start?**
- Make sure `.env` has `PREDICTIONGUARD_API_KEY` and `PREDICTIONGUARD_URL`
- Check dependencies: `pip install -r backend/requirements.txt`

**Frontend won't connect?**
- Make sure backend is running on port 8000
- Check browser console for CORS or network errors

**Generation fails?**
- Check `backend_logs/<job_id>.log` for detailed error info
- Verify YouTube URL is valid and accessible
